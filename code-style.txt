Spacing:
  1 Tab / 4 spaces
  
Function Names and Class Names:
  CapitalCase
  
Variable Names and Method Names:
  camelCase

Error Handling:
  try:
    code-that-may-throw-error()
  except SpecificError as error:
    raise SpecificError("Personal Error Statement")

Line Length:
  Preferably less than 80, max of 100
  
Commenting:
  Comment blocks of codes, try to refrain from repetitive comments/comments for every line

Bad Style:
  Variables with random letters, one letter, more than three words
  One line functions (that are convoluted)
  Making a class for one function
  Naming things with one letter difference
